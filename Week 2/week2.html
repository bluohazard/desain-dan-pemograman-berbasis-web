Pengenalan HTML

Apa itu HTML ?

  1. Singkatan dari Hypertext Markup Language
  2. Dikenal sebagai bahasa markup atau formating language ditandai dengan penggunaan tag untuk meng-markup konten didalamnya
    Contoh : <html></html>
  3. Setiap tag pasti berpasangan, pembuka (<html>) dan penutup (<html>). Penutup tag ditandai dengan "/ " pada bagian depan nama tag
  4. HTML Language != Programming Language, why ?

Pengenalan HTML

Struktur HTML

  1. <!DOCTYPE html>
    Menginformasikan kepada browser bahwa dokumen merupakan dokumen HTML
  2. <html>..</html>
    Inisiasi awal dokumen HTML. Seluruh dokumen html harus diawali dan ditutup dengan tag ini
  3. <head>..</head>
    Digunakan untuk menyimpan informasi berkaitan dengan dokumen HTML. Contoh : penggunaan tag <title></title> untuk memberikan judul pada dokumen HTML. Informasi yang ada dalam markup <head>..</head> tidak akan ditampilkan
  4. <body>..</body>
    Merupakan bagian utama dari dokumen HTML dan digunakan untuk menyimpan semua konten HTML yang akan ditampilkan ke dalam Browser
<!DOCTYPE html>
  <!-- Comment Tag -->
  <html>
  <head>
    <title>Ini Judul</title>
  </head>
  <body>
   Ini body
  </body>
  </html>

Memuat Gambar #1

  1. Diperlukan minimal tag <img /> dengan atribut src untuk menampilkan gambar pada dokumen HTML
  2. Terdapat atribut lain yang dapat digunakan pada tag <img /> seperti alt, hight, widht
  3. Atribut alt digunakan untuk memberikan alternatif tampilan teks jika gambar gagal ditampilkan oleh browser
  4. height digunakan untuk mengatur panjang dari gambar dan widht untuk mengatur lebar dari gambar. Ukuran yang digunakan dalam satuan pixel (px)
  5. NOTE : Jika pada tag <img /> tidak menggunakan opsi atribut heigh dan/atau widht, maka gambar akan dimuat dengan menggunakan ukuran asli
  6. Praktikkan dengan menulis kode di samping!
<!DOCTYPE html>
<html>
<head>
  <title>Load Image in HTML</title>
</head>
<body
  <h1>Welcome to HTML</h1>
  <p>your name</p>
  <img src="img.png" alt="Ini gambarku" widht="240" hight="320" />
</body>
</html>

Membuat Tautan / Link

Klik disini untuk mendownload ?

  1. Untuk membuat sebuah tautan diperlukan tag <a></a>
  2. Diperlukan juga atribut href untuk menyimpan informasi url yang akan dituju
  3. Teknik penulisan Path atau URL dalam href serupa dengan src
  4. Terdapat juga atribut target="_blank" untuk membuka tautan pada window atau tab yang baru
<!DOCTYPE html>
<html>
<head>
  <title>Links in HTMLL</title>
</head>
<body
  <h1>Create Some Links</h1>
  <!-- Link Pada File HTML tertentu -->
  <a href="file2.html">Menuju File 2</a>
  <!-- Membuka Tautan Web -->
  <a href="http://polinema.ac.id">Menuju Web Polinema</a>
</body>
</html>

Membuat List #1

List merupakan tag yang digunakan untuk membuat sebuah daftar. Terdapat 3 jenis list dalam HTML, yaitu Ordered List <ol></ol>, Unordered List <ul></ul>, dan Data List <dl></dl>

  1. <ol></ol> digunakan untuk membuat daftar dengan penomoran otomatis
  2. <ul></ul> digunakan untuk membuat daftar dengan tanpa penomoran, namun menggunakan penanda bullet
  3. <dl></dl> tipe daftar / list khusus yang dapat digunakan seperti untuk membuat daftar istilah pada kamus
  4. Setelah menginsiasi jenis list yang digunakan, untuk memasukkan daftar item kedalam list tersebut digunakan tag <li></li> didalam tag list
  5. Perhatikan contoh pada code snippet disamping
<!DOCTYPE html>
<html>
<head>
  <title>Ordered and Unordered List</title>
</head>
<body
  <ol>
    <li>Ordered list item 1</li>
    <li>Ordered list item 2</li>
  </ol>
  <ul>
    <li>Unordered list item 1</li>
    <li>Unordered list item 2</li>
  </ul>
</body>
</html>

ID vs Class

Id dan class merupakan atribut yang ada pada setiap elemen atau tag HTML. Apa perbedaan keduanya ?

  1. Id digunakan untuk berikan pengenal pada tag HTML. Id bersifat UNIQUE, sehingga tidak boleh ada 2 tag dengan Id yang sama
  2. Class digunakan untuk tag HTML dengan karateristik yang sama
  3. Id dan Class dapat digunakan sebagai penanda pada saat melakukan styling dengan menggunakan CSS (dibahas selanjutnya)
  4. Id dapat digunakan sebagai penanda manipulasi data maupun tag dengan menggunakan JavaScript (JS)
<!DOCTYPE html>
<html>
<head>
  <title>The DIV</title>
</head>
<body>
<div id="header">
  Header
</div>
<div id="navigation">
  Navigation
</div>
<div id="main-content">
  <div class="blok">
  </div">
  <div class="blok">
  </div">
</div>
</body>
</html>

Pengenalan HTML

Parent, Child, Sibling

Apa hubungan ketiganya ?

  1. <div id="wrap"> merupakan induk dari dari tag-tag yang ada didalamnya, sehingga dapat disebut sebagai parent
  2. <div id="main-content"> merupakan child dari <div id="wrap"> karena berada didalam tag <div id="wrap">. Namun <div id="main-content"> juga sebagai parent bagi <div id="content"> dan <div id="sidebar"> karena keduanya terdapat didalamnya
  3. <div id="content"> dan <div id="sidebar"> merupakan sibling karena berada pada tingkatan yang sama
<!DOCTYPE html>
<html>
<head>
  <title>The DIV</title>
</head>
<body>
<div id="wrap">
  <div id="main-content">
    <div class="blok"></div">
    <div class="blok"></div">
    </div>
</div>
</body>
</html>

Tabel #1

Bagaimana cara membuat tabel ?

  1. Untuk membuat sebuah tabel diperlukan tag <table></table>
  2. Didalam sebuah tabel tentunya terdapat baris dan kolom. <tr></tr> digunakan untuk membuat baris, sedangkan <td></td> digunakan untuk membuat kolom
<!DOCTYPE html>
<html>
<head>
  <title>Table</title>
</head>
<body>
<table border="1">
  <!-- Baris Pertama -->
  <tr>
    <td>NIM</td>
    <td>Nama</td>
    <td>Jenis Kelamin</td>
  </tr>
  <!-- Baris Kedua -->
  <tr>
    <td>NIM</td>
    <td>Nama</td>
    <td>Jenis Kelamin</td>
  </tr>
</table>
</body>
</html>

Pengenalan HTML

Tabel #2

Struktur yang dianjurkan

  1. Tabel tentunya mempunyai header untuk meberikan nama pada setiap kolom. Untuk membuatnya digunakan tag <thead></thead>
  2. Untuk badan tabel, dimana berisi informasi atau data, digunakan <tbody></tbody>
  3. Terdapat tag <tfoot></tfoot> untuk memberikan informasi tambahan pada bagian bawah tabel
<!DOCTYPE html>
<html>
<head>
  <title>Table</title>
</head>
<body>
  <table border=1>
    <thead>
      <tr>
        <td>NIM</td>
        <td>Nama</td>
        <td>Jenis Kelamin</td>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>5210100016</td>
        <td>Muhammad Afif Hendrawan</td>
        <td>Laki-laki</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <th colspan="3">Politeknik Negeri Malang</th>
      </tr>
    </tfoot>
  </table>
</body>
</html>
