Penganalan Bootstrap

Apa itu Bootstrap ? #1



  1. Merupakan library yang digunakan untuk mempermudah membangun desain front-end
  2. Dikembangkan oleh Mark Otto dan Jacob Thornton dari Twitter
  3. Merupakan kombinasi HTML, CSS, dan JS yang dapat digunakan untuk kebutuhan desain tipografi, form, button, navigasi, dsb.
  4. Versi terbaru dari Bootstrap saat ini adalah versi v4.1.0

Contoh Tema dengan Bootstrap

Bootstrap Grid #1

Konsep Grid Pada Bootstrap

  1. Merupakan sistem 12 kolom yang digunakan Bootstrap dalam mendefinisikan ukuran komponen
  2. Grid sistem juga memiliki beberpa opsi untuk menentukan lebar maksimal, yaitu : Extra small, Small, Medium, Large, Extra large
  3. Prefix yang digunakan dapat dilihat pada tabel
  4. Referensi : http://getbootstrap.com/docs/4.1/layout/grid/

Studi Kasus

Problem #1 - Header

  1. Bagian paling atas dari desain pada studi kasus adalah sebuah header
  2. Tag HTML <header></header> dapat digunakan untuk membuat sebuah header
  3. Didalam header buatlah sebuah tag nav dengan id="webBrand" dan class .navbar
  4. NB : Mengapa menggunakan <nav></nav>, dan <header></header> dan tidak div ? Jawaban semantic = sesuai peruntukan
  5. Berikan efek warna gray dengan menambahkan class .navbar-dark dan .bg-dark
  6. Tambakan anchor a dengan class .navbar-brand didalam nav

Pengenalan Bootstrap

Studi Kasus

Problem #2 - Navigation

  1. Buat kembali element nav dan letakkan dibawah nav "webBrand"
  2. Gunakan class .navbar navbar-expand-lg navbar-dark bg-secondary pada nav
  3. Class .navbar-expand-{ sm | md | lg | xl } digunakan sebagai wrapping atau ruang untuk meletakkan menu navigasi web
  4. Didalam navbar buatlah sebuah div dan berikan class .collapse navbar-collapse
  5. Didalam div buat unordered list dengan class .navbar-nav
  6. Untuk setiap list item gunakan class .nav-item
  7. Didalam list item, gunakan anchor sebagai item dengan class .nav-link

Problem #3 - Main Content dan Sidebar

  1. Untul membuat bagian main content dan sidebar, ingat kembali konsep grid
