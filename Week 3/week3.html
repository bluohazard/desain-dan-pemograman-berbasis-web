Komponen CSS ? #1

Apa saja komponen utama pada CSS?

  1. Terdiri dari 3 komponen utama, Selector, Property, dan Value.
  2. Selector digunakan sebagai penanda elemen/tag HTML yang akan di-styling. Selector dapat digunakan langsung pada tag HTML atau spesifik pada ID atau Class
  3. Simbol hashtag # digunakan oleh selector untuk merujuk pada ID terntentu. Sedangkan simbol titik . digunakan untuk class.
  4. h1, #judul, dan .title merupakan contoh selector

Pengenalan CSS

Komponen CSS ? #2

Apa itu font-size dan 20px ?

  1. font-size adalah sebuah property yang memberikan sifat-sifat dari selector. Sifat dapat bermacam-macam, seperti warna font, ukuran font, jarak antar elemen, garis, dsb
  2. 20px adalah nilai dari property yang kita gunakan
  3. property dan value dipisahkan oleh simbol titik dua : dan setiap inisiasi property harus diakhiri dengan simbol titik koma ;

CSS Shorthand

Lelah menuliskan -top, -right, -bottom, dan -left ? We need a solution !

  1. Kita dapat menuliskan 4 value, margin: 10px 20px 5px 30px; yang berarti ukurang top 10px, right 20px, bottom 5px, dan left 30px. Ingat urutannya adalah top, right, bottom, left !
  2. Kita dapat menuliskan 3 value, margin: 10px 20px 5px;. Sesuai dengan urutan berarti top, right, bottom. left akan mengikuti ukuran dari right yaitu 20px
  3. Kita dapat menuliskan 2 value, margin: 10px 20px, maka sesuai urutan, bottom akan berukuran 10px dan left akan merukuran 20px
  4. Kita dapat menuliskan 1 value, margin : 5px;, yang berarti semua sisi akan berukuran sama 
.box1 {
  margin: 10px 20px 5px 30px;
}
.box2 {
  margin: 10px 20px 5px;
}
.box3 {
  margin: 10px 20px;
}
.box4 {
  margin: 5px;
}

Typography

Bagaimana memberikan style pada "tulisan" ?

  1. Tedapat banyak jenis font yang dapat digunakan dalam pengembangan web. Default Times New Roman
  2. Akan tetapi, pastikan browser mengenali font yang digunakan. Contoh font Helvatica hanya dapat di-load pada browser Safari
  3. font-family digunakan untuk menentukan jenis font yang digunakan
  4. Contoh : font-family:calibri;
  5. Jika font-family:calibri, serif, arial, verdana;, maka urutan font menunjukkan prioritas penggunaan
  6. font-size digunakan untuk menentukan ukuran font. Satuan yang dapat digunakan yaitu, px, point (pt), dan em
  1. font-weight digunakan untuk menentukan ketebalan font
  2. letter-spacing digunakan untuk menentukan jarak antar karakter
  3. line-height digunakan untuk jarak antar baris
  4. text-align digunakan untuk menjadikan teks rata kiri, kanan, tengah, atau rata kanan-kiri

CSS Images #1

How to put an images in CSS

  1. Untuk memuat gambar pada sebuah elemen dengan menggunakan CSS diperlukan property background
  2. Kita juga dapat melakukan perulangan penggunaan gambar dengan property backgroud-repeat dan value repeat-x untuk perulangan thd sumbu x, repeat-y untuk sumbu y
  3. Terdapat juga value no-repeat untuk gambar tidak ditampilkan berulang
  4. Untuk mengatur posisi digunakan background-position-x untuk sumbu x, background-position-y untuk sumbu y, dan background-position untuk mengatur keduanya sekaligus
.logo {
  /* Load Gambar */
  background: url('../sampel-code/img/logopolinema.png');
  background-repeat: no-repeat;
  background-size: 138px 845px;
  background-position: 5px 5px;
}

Floating #2-3

Permasalahan dalam floating

  1. Mengapa #main-content dan sidebar tidak masuk didalam #content ?
  2. Mengapa tinggi #content tidak setinggi #main-content + padding 20px pada top dan bottom ?
  3. Hal ini karena terdapat property floating pada #main-content dan #sidebar sehingga #content menganggap tidak ada elemen didalamnya

Solusi ?

Tambahkan property overflow dengan value hidden pada #content

#content {
  ...
  overflow: hidden;
}

Kedua, tambahkan div kosong sebagai child terakhir dari #content pada file HTML

<div id="content">
  ...
  <div style="clear:both"></div>
</div>

Positions

Jenis-jenis pengaturan posisi elemen dalam HTML

  1. Static, dimana kita tidak bisa menentukan posisi elemen HTML. Posisi static tidak memerlukan penggunaan property
  2. position:relative digunakan untuk mengatur posisi elemen sesuai dengan koordinat yang diinginkan dengan menggunakan property tambahan top, right, bottom, dan/atau left
  3. position:relative akan sangat dinamis berdasarkan ukuran tampilan browser serta efek zooming
  4. position:absolute digunakan agar posisi elemen child tidak bergantung pada elemen parent
  5. position:fixed digunakan untuk menjadi posisi elemen menjadi tetap dan tidak dipengaruhi ukuran tampilan browser maupun efek zooming
