Penganalan jQuery

Apa itu jQuery ?



  1. Merupakan library JS
  2. Dapat digunakan untuk manipulasi HTML dan CSS (DOM) dengan lebih mudah
  3. Dapat digunakan untuk membuat efek dan animasi sehingga mempercantik UI dan memberikan UX yang lebih baik
  4. Dapat digunakan untuk melakukan teknik AJAX (Asynchronous JavaScript and XML) dengan lebih mudah
  5. AJAX merupakan teknik komunikasi client dengan server (ex: request dan receive) tanpa harus me-reload. Proses ini berjalan pada sistem backgroud atau lebih dikenal sebagai proses asynchronous

Pengenalan jQuery

How to use jQuery library


  1. Terdapat dua cara menggunakan library jQuery
  2. Pertama dengan mengunduh library secara langsung dari https://jquery.com/download/
  3. Stable release adalah versi dengan keterangan production
  4. Terdapat 2 versi production, full dan slim. Versi slim merupakan library jQuery tanpa adanya fitur AJAX handling dan efek animasi
  5. Kedua dengan menggunakan content delivery network (CDN)
  6. Dengan CDN kita hanya perlu "memanggil" URL dari library jQuery
  7. Terdapat 4 layanan CDN yang menyediakan CDN jQuery : Google, Microsoft, CDNJS, jsDelivr
  8. Perhatikan potongan kode disamping

Pengenalan jQuery

Syntax jQuery #1

Bagaimana cara menggunakan jQuery ?

  1. Syntax dasar dari jQuery adalah $(selector).action()
  2. Dollar sign ($) menandakan penggunaan fungsi dari library jQuery
  3. Selector merupakan element DOM yang akan dimanipulasi dengan menggunakan jQuery
  4. action() adalah fungsi jQuery yang digunakan
  5. Perhatikan contoh pada potongan kode disamping!
// Menyembunyikan elemen DOM "p"
$("p").hide();
// Menampilkan elemen dengan id "mainPage"
$("#mainPage").show();
// Menjalankan fungsi ketika DOM telah termuat sempurna
$(document).ready(function(){
  //some function
});

Syntax jQuery #2

jQuery selector

  1. Sama seperti CSS dan JS, jQuery juga menggunakan selector untuk memilih element DOM yang akan dimanipulasi
  2. Konsep selector seperti tag, class, id, parent, dan child harus dipahami untuk memudahkan saat akan melakukan manipulasi DOM dengan menggunakan jQuery
  3. Gambar disamping merupakan contoh penggunaan selector pada jQuery

Pengenalan JS

jQuery Events #1


  1. Terdapat banyak event pada jQuery
  2. Daftar event jQuery dapat dilihat di https://api.jquery.com/category/events/
  3. Perlu diingat, dari beberapa event yang terdapat dari dokumentasi jQuery diatas, terdapat beberapa event dengan status deprecated
  4. Deprecated adalah istilah yang digunakan (dalam hal ini fungsi atau method) yang tidak lagi digunakan atau hilang dalam versi terbaru dari software/aplikasi/tools/library
  5. Event pada jQuery diantaranya : .click(), .change(), .keypress(), .submit()

Penanganan form dengan jQuery

jQuery Form #2
