Penganalan PHP

Apa itu PHP ?



  1. Merupakan bahasa pemograman server-side yang didesain untuk pengembangan aplikasi web
  2. Merupakan bahasa pemograman web yang masih sangat populer hingga saat ini
  3. Karena merupakan bahasa pemograman server-side maka seluruh peprosesan kode dilakukan di dalam server
  4. Client hanya akan melihat hasil dari kode-kode PHP yang telah selesai diproses pada server

Variable dan Konstanta pada PHP

Variable

  1. Variable pada PHP diawali dengan dollar sign atau $
  2. Penamaan variable bersifat case sensitive
  3. Atauran lazim yang berlaku untuk penamaan variable dalam PHP adalah menggunakan camelCase atau dipisah dengan _ jika lebih dari satu kata
  4. NAMA VARIABLE TIDAK BOLEH DIAWALI DENGAN ANGKA
  5. NAMA VARIABLE TIDAK BOLEH MENGANDUNG KARAKTER SPASI
  6. Nilai dalam variable dapat diubah
  7. Tipe data dari variable dalam PHP tidak perlu dideklarasikan sebelumnya

Konstanta

  1. Konstanta merupakan nilai variable yang tidak dapat dirubah atau dimodifikasi
  2. Pembuatan konstanta dapat dilakukan dengan cara penambahan keyword const atau penggunaan fungsi define()
  3. Contoh dengan const : const kota = "Malang";
  4. Contoh dengan define() : define("kota","Malang");
  5. Teknik penamaan konstanta dapat mengikuti teknik penamaan variable

Latihan #1

  1. Buatlah kode program yang bisa membedakan angka genap dengan angka ganjil. Misalkan variabel $a berisi 50, akan tampil “50 adalah angka genap. Jika $a berisi 7, maka akan tampil “7 adalah angka ganjil”.
  2. Tips : gunakan operator modulus (%)

Latihan #2

Buatlah sebuah kode program yang terdiri dari 2 variabel: $a dan $b. Kedua variabel ini berisi angka integer atau float. Bandingkan dinilai keduanya sehingga tampilan akhir adalah salah satu kondisi berikut:

  1. $a lebih kecil dari $b
  2. $a lebih besar dari $b
  3. $a sama besar dengan $b

Latihan #3

Buatlah kode program yang akan memeriksa apakah $username berisi string ”JTI” dan $password berisi string “polinema”. Jika benar, tampilkan pesan: “Username dan Password sesuai, hak akses diberikan”. Jika salah, tampilkan pesan: “Username atau Password tidak sesuai!”.

Latihan #4

Modifikasi kode dari Latihan #3 sehingga menghasilkan handling untuk kondisi-kondisi berikut :

  1. Username sesuai, password sesuai: ”Username dan password sesuai, hak akses diberikan”
  2. Username sesuai, password tidak sesuai: ”Username sesuai, password tidak sesuai!”
  3. Username tidak sesuai, password sesuai: ”Username tidak sesuai, password tidak sesuai!”
  4. Username tidak sesuai, password tidak sesuai: “Nama dan Password tidak sesuai!”
