Penganalan JavaScript

Apa itu JavaScript (JS) ?

  1. JavaScript != Java
  2. JS merupakan bahasa pemograman yang dinamis
  3. JS juga merupakan bahasa scripting yang menjadi bagian dari ECMAScript (ES)
  4. ES merupakan standar internasional untuk scripting languange berdasarkan spesifikasi ECMA-262 dan ISO/IEC 16262
  5. Standar terbaru dari ES adalah ES6. More about ES6

Mengapa belajar JavaScript ?

  1. One of 3 web developer must learn! (selain HTML dan CSS)
  2. JS is lightweight, interpreted programming languange
  3. Terintegrasi dengan HTML dan CSS
  4. Commonly used as client side script
  5. Thats all ???

var pizzaTopping = new Array();
pizzaTopping[0] = 'Jamur';
pizzaTopping[1] = 'Keju';
pizzaTopping[2] = 'Smooked beef';

Form Handling

Form Validation #1

  1. Didalam sebuah form terkadang terdapat isian yang harus alias wajib diisi
  2. Sebagai programmer yang baik, kita harus memperhatikan hal ini dan memastikan user melakukan isian dengan benar
  3. Salah satu yang dapat dilakukan adalah validasi form
  4. Contoh validasi yang paling sederhana adalah memvalidisi isian form, apakah sudah diisi atau belum.
  5. Perhatikan kode disamping!
function validateForm () {
  var fname = document.forms["myForm"]["fname"].value;
  var lname = document.forms["myForm"]["lname"].value;

  if (fname == "" && lname == "") {
      alert('Nama depan dan belakang harus di isi!');
      return false;
  } else if (fname == "") {
      alert('Nama depan harus di isi!');
      return false;
  } else if (lname == "") {
      alert('Nama belakang harus di isi!');
      return false;
  } else {
      return true;
  }
}
