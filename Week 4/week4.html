Penganalan JavaScript

Apa itu JavaScript (JS) ?

  1. JavaScript != Java
  2. JS merupakan bahasa pemograman yang dinamis
  3. JS juga merupakan bahasa scripting yang menjadi bagian dari ECMAScript (ES)
  4. ES merupakan standar internasional untuk scripting languange berdasarkan spesifikasi ECMA-262 dan ISO/IEC 16262
  5. Standar terbaru dari ES adalah ES6. More about ES6

Mengapa belajar JavaScript ?

  1. One of 3 web developer must learn! (selain HTML dan CSS)
  2. JS is lightweight, interpreted programming languange
  3. Terintegrasi dengan HTML dan CSS
  4. Commonly used as client side script
  5. Thats all ???

Apa itu DOM

  1. Singkatan dari Document Object Model atau lebih dikenal dengan HTML DOM
  2. HTML DOM akan dibuat ketika sebuah web pages telah dimuat (loaded) oleh web browser
  3. Dengan adanya HTML DOM, memungkikan JS untuk mengakses element HTML bahkan melakukan manupulasi element HTML secara langsung
source: https://www.w3schools.com/js/pic_htmltree.gif

Pengenalan JS

Mengakses element DOM #1

  1. Dalam lingkup DOM, element HTML didefinisikan sebagai obyek
  2. HTML DOM method merupakan action yang kita bisa lakukan pada element HTML. Ex: membuat atau membuang elemen HTML
  3. HTML DOM properties merupakan value yang bisa kita dapatkan (get) dan/atau bisa kita rubah (set)
  4. getElementById merupakan metode paling lazim digunakan untuk mengakses obyek HTML DOM
  5. Perhatikan contoh disamping!
<html>
  <head>
    <title>Latihan JS 1</title>
  </head>
  <body>
    <p id="demo"></p>
    <script>document.getElementById("demo").innerHTML = "Hello World";</script>
  </body>
</html>

Apa itu event dalam JS ?

  1. Dengan menggunakan HTML DOM dan JS, element HTML dapat bereaksi sesuai dengan tujuan yang diinginkan. Inilah yang disebut sebagai event
  2. Eventdiantaranya dapat berupa kejadian ketika halaman web selesai dimuat, perubahan pada input, ketika tombol diclick, dsb
  3. Practice makes perfect!Buat sebuah folder bernama js, salin script disamping, dan simpan dengan nama script1.js
function getValue() {
  var firstName = document.getElementById('firstName').value;
  var lastName = document.getElementById('lastName').value;
  var par = document.createElement('p');
  var node = document.createTextNode('Hallo ' + firstName + ' ' +lastName);
  par.appendChild(node);
  document.getElementById('div').appendChild(par);
}

function showPrompt() {
  var fullName = prompt('Please enter your full name!');
  var par = document.createElement('p');
  if (fullName != null) {
    var node = document.createTextNode('Hallo ' + fullName + '!');
    par.appendChild(node);
    document.getElementById('window').appendChild(par);
  }
}

Array #1

Masih ingat ???

  1. Terdapat 3 cara untuk membuat array dalam JS, pertama dengan menggunakan indexer, kedua dengan condensed array, dan yang terakhir adalah literal array
  2. Perhatikan contoh disamping
// Indexer
var pizzaTopping = new Array();
pizzaTopping[0] = 'Jamur';
pizzaTopping[1] = 'Keju';
pizzaTopping[2] = 'Smooked beef';

// Condensed array
var pizzaTopping = new Array('Jamur', 'Keju', 'Smooked beef');

// Literal array
var pizzaTopping = ['Jamur', 'Keju', 'Smooked beef'];
